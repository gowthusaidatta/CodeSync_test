🚀 CodeSync – Complete Feature List

✅ Core Functional Features

Feature Description
🔢 Coding Score Tracker Fetches scores from CodeChef, LeetCode, HackerRank, and GFG via Lambda REST API
🧠 Smart Profile Management Users can enter or auto-fetch usernames on sign-in; stored in DynamoDB
🌓 Dark Mode UI Sleek Tailwind CSS frontend with light/dark mode support
📈 Live Score Dashboard Displays platform-wise scores with animations and icons

🔐 Authentication & User Management

Feature Description
🔐 AWS Cognito Login/Signup Email-password based login/signup with hosted UI
🧾 Token Expiry Handling 60-minute access tokens, 5-day refresh tokens using SRP authentication
🧠 Remember Usernames Saves platform usernames after first login and auto-fills next time
🚪 Logout Support Redirects to sign-out URL using Cognito Hosted UI

📤 Notifications & Engagement

Feature Description
📩 SNS Email Reminders Sends email reminders for inactive coders via Amazon SNS
⏰ Cron Job via EventBridge Daily 12:30 PM UTC Lambda trigger to check inactivity
🧮 NoCodingDetected Table Tracks users with no activity; cleaned after 22 hours
🧪 Verified Email Check Sends emails only if user has verified SNS subscription

📊 Analytics & Dashboards

Feature Description
📊 Amazon QuickSight Integration (Planned) Advanced visual dashboards for coding progress
🥇 Leaderboard System (Planned) Compare scores of registered users
🤖 AI-Based Coding Planner (Planned) Recommends targets based on previous performance

Stack Component Usage
